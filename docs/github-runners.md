## Useful GitHub Runner Commands

### Start Runner

```bash
sudo ./svc.sh start
```

### Stop Runner

```bash
sudo ./svc.sh stop
```

### Restart Runner

```bash
sudo ./svc.sh restart
```

### Check Runner Status

```bash
sudo ./svc.sh status
```

### Install Runner

```bash
./config.sh --url <repository-url> --token <runner-token>
```

### Uninstall Runner

```bash
sudo ./svc.sh uninstall
```

### Update Runner

```bash
sudo ./svc.sh update
```
