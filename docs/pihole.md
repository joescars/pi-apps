# Pihole Helpful Info

## Useful Pihole Commands

### Check Pihole Status

```bash
pihole status
```

### Restart Pihole

```bash
pihole restartdns
```

### Update Pihole

```bash
pihole -up
```

### Enable Pihole

```bash
pihole enable
```

### Disable Pihole

```bash
pihole disable
```

### View Pihole Logs

```bash
pihole -t
```

### Flush Pihole Logs

```bash
pihole flush
```

### Add a Domain to Blocklist

```bash
pihole -b <domain>
```

### Remove a Domain from Blocklist

```bash
pihole -b -d <domain>
```
